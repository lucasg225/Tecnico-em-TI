Componentes do Computador

Projeto TI V5

Principais Componentes

🧠

Processador (CPU)

Executa instruções e coordena todas as operações do computador.

💾

Memória RAM

Armazena temporariamente os dados utilizados pelos programas em execução.

🎮

Placa de Vídeo (GPU)

Processa gráficos, imagens e vídeos.

🗄️

SSD / HD

Responsável pelo armazenamento permanente de arquivos e do sistema.

🔌

Fonte de Alimentação

Converte energia elétrica e alimenta todos os componentes.

🖥️

Placa-Mãe

Interliga todos os componentes do computador.
